BimE.IDTools

Un set di strumenti pensato per migliorare l'interazione con gli ID degli elementi Revit. Rapido, leggero e funzionale.

  • ID Viewer: mostra l’ID univoco degli oggetti selezionati.
  • ID Copy: consente la copia dell’ID di uno o più oggetti selezionati.
  • Select by ID: seleziona automaticamente elementi partendo da un elenco di ID.
