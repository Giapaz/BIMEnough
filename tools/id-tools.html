Why ID Tools Are Essential

Because finding elements shouldn't be like finding a needle in a haystack

ID Viewer

See element IDs and unique IDs for any selection instantly. Perfect for debugging, coordination, or understanding your model structure. Copy IDs to clipboard with one click!

ID Copy

Copy element IDs to parameters automatically. Perfect for creating unique element tags, coordination references, or linking to external databases. Works with any text parameter!

Select by ID

Paste a list of element IDs and select them all instantly. Perfect for troubleshooting, coordination workflows, or working with external data that references specific elements.

Both ID Types

Work with both Element IDs (numbers) and Unique IDs (GUIDs). Switch between them based on your needs. Unique IDs are perfect for external references and never change!

Excel Integration

Export element information with IDs to Excel for analysis or coordination. Perfect for creating element databases or sharing with consultants who need specific element references.

Highlight & Navigate

Highlight selected elements in the model and navigate to them instantly. No more hunting through views to find that one specific element!
